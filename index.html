How It Works

1. Swing Detection

Detects swing highs and lows using configurable swing strength and lookback.

2. Projection Analysis

Combines multiple methods including Fibonacci, cycle, and measured moves to project future pivot zones.

3. AutoTune & Learning

Adaptive settings based on market regime and ongoing learning from realized outcomes.

4. Visual Overlays

Draws vertical pivot lines, price levels, and directional arrows with customizable colors and sizes.

FAQ

What does this indicator do?

Predicts future pivot confluence zones using cycle, swing, Fibonacci, and multi-timeframe analysis.

Is it compatible with NinjaTrader 8?

Yes, fully compatible and designed specifically for NinjaTrader 8.

Can I customize the learning parameters?

Yes, all learning and AutoTune parameters are configurable.

Does it repaint?

No, it uses confirmed swing points and historical data for projections.

Can I use it for multiple timeframes?

Yes, it supports multi-timeframe analysis with multiple data series.

How do I interpret the score panel?

The score panel breaks down the weighted contributions of different methods to the projection score.
